# Workspace Demo

The simulated workspace (`/workspace` on the bubble server): openable demo
drawings (real PDFs in `demo/drawings/`) over the same [[Ambient Trigger]]
matcher and the same live store — so judges experience the ambient nudge
from a browser, zero install. Labeled honestly: the drawings are demo files;
everything Trace does is live.

## Related

- Uses [[Ambient Trigger]].
